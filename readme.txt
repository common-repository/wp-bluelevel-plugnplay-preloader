== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/wp-bluelevel-plugnplay-preloader` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP BlueLevel PlugNPlay Preloader screen to configure the plugin
4. And you're done! No coding needed!
